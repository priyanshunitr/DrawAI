# DrawAI Frontend

Vite, React, and TypeScript frontend for the DrawAI diagramming and documentation product.

## Stack

- Build tool: Vite
- UI: React
- Language: TypeScript
- Icons: lucide-react
- Styling: plain CSS modules by route/component file

## Local Setup

```bash
npm install
npm run dev
```

Useful scripts:

- `npm run dev` starts the local Vite server.
- `npm run typecheck` runs TypeScript checks.
- `npm run build` creates a production build.
- `npm run preview` serves the production build locally.
